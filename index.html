이모티콘 → 게임, 같은 파이프라인의 확장

지난 이모티콘 편에서 쓴 held-cel 방식을 게임 액션에 그대로 적용했습니다. 아래 스프라이트는 전부 실제 생성물이에요.

01

Clean ref

만화 컷 → GPT Image 2로 단독 클린 레퍼런스 추출.

02

Held-cel

액션별 16프레임 시트 — 몸통 고정, 필요한 부분만 움직임.

03

Stabilize

셀 분할·투명화 후 발 baseline·크기를 코드로 재정렬해 흔들림 제거.

04

Atlas

안정화 프레임을 균일 셀 투명 시트로 재패킹.

05

Unity

오토슬라이스 .meta + frames.json 동봉 — 드래그로 임포트.

이 데모의 스프라이트시트와 Unity 번들은 브라우저에서 실제로 생성됩니다. 캐릭터·프레임은 GPT Image 2로 만든 실제 held-cel 결과물이고, 시뮬레이션되는 것은 "생성 진행" 연출뿐이에요. 박사·개발자는 게임용 최종 공정 진행 중입니다.
