# Security Policy

## Supported versions
This project ships from `main`. Fixes land on `main`.

## Reporting a vulnerability
Please do **not** open a public issue for a security problem.
- Preferred: private report via GitHub Security Advisories ("Report a vulnerability" under Security).
- Or email: thakiaiplatform@gmail.com

Include what you found, how to reproduce it, and the impact. You will get an acknowledgement within a few days.

## Notes
- Keep credentials in environment variables, never in committed files.
- Secret scanning and push protection are enabled on this repository.
